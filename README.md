node-mired
==========

Convert between Mired (micro reciprocal degree) and temperature

https://en.wikipedia.org/wiki/Mired

Usage
-----

``` js
var mired = require('mired');

mired.kelvinToMired(2700)
// => 370.3703703703704

mired.miredToKelvin(370)
// => 2702.7027027027025
```

Install
-------

    npm install mired
